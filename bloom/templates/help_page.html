1. Question Page

The Question Page is designed to help faculty create detailed question papers efficiently. Here's how you can use it:

  • Select a Course: Use the dropdown menu to choose the course for the question paper.
  • Enter Total Marks: Input the total marks for the entire question paper.
  • Add Sections: Click the "Add Section" button to divide the paper into multiple sections.
  • Enter Marks for Questions: Specify the marks allocated for each question in a section.
  • Mandatory Questions: Use the checkbox to mark questions that are mandatory to attempt.
  • Add Multiple Sections: Flexibly add or remove sections based on your requirements.
2. Bloom's Levels Page

The Bloom's Levels Page provides CRUD functionality for managing Bloom's taxonomy levels and verbs. Here's what you can do:

  • View Bloom's Levels: See a comprehensive list of all Bloom's taxonomy levels and their associated verbs.
  • Add New Level: Click the "Add Bloom's Level" button to define a new level with verbs.
  • Edit Existing Levels: Use the icon to update a Bloom's level or its verbs.
  • Delete Levels: Use the icon to remove a level permanently.
3. Course Outcomes Page

The Course Outcomes Page allows faculty to manage course outcomes for each course:

  • View Course Outcomes: Displays all the outcomes related to every course in a structured table.
  • Add Course Outcome: Select a course and add a specific outcome to it.
  • Edit Course Outcomes: Click the icon to modify existing course outcomes.
  • Delete Course Outcomes: Use the icon to delete an outcome.
4. Program Outcomes Page

The Program Outcomes Page manages program-level outcomes across courses:

  • View Program Outcomes: Displays all program outcomes for all courses.
  • Add Program Outcome: Select a course and define a program outcome for it.
  • Edit Program Outcomes: Use the icon to update existing program outcomes.
  • Delete Program Outcomes: Use the icon to remove a program outcome.
